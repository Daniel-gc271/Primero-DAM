Listas

Las listas en html permiten ordenar cosas, existen listas ordenadas y desordenadas.
Se construyen con las etiquetas <ol>, <ul> y <li>.
Tambien existe un tipo de listas especiales, llamadas listas de definiciones.
Se construyen con las etiquetas <dl>, <dt> y <dd>.

Listas ordenadas

"ol" Sirve para listas ordenadas, las cuales automaticamente generan un indice.
Los contenidos se declaran mediante la etiqueta <li>.
Por ejemplo:

  1. Patata
  2. Zanahoria
  3. Tomate
  4. Lechuga
  5. Brocoli

Aqui tenemos una lista ordenada con 5 elementos

Listas desordenadas

<ul> sirve para las listas desordenadas, las cuales carecen de numeracion.
Los contenidos se declaran mediante la etiqueta <li>.
Por ejemplo:

  • Patata
  • Zanahoria
  • Tomate
  • Lechuga
  • Brocoli

Aqui tenemos una lista con los mismos elementos que la anterior pero sin numeracion

Listas de definiciones

Las listas de definiciones sirve para agrupar definiciones de manera similar a un diccionario.
se inicializan con la etiqueta <dl>, <dt> indica el termino a definir y <dd> contiene la definicion propia.
Por ejemplo:

dl
Etiqueta que inicializa la lista de definiciones
dt
Etiqueta que indica el termino que se va a definir
dd
Etiqueta donde va la definicion propia del termino
Bonus

Se pueden anidar ambos tipos de listas.
Por ejemplo:

  • Patata
  • Zanahoria
  • Tomate
    1. Lechuga
    2. Brocoli
