Links en html

Los links en html permiten interconectar webs separadas para organizarlas en distintos archivos/carpertas html

Índice
Explicacion

Existen dos tipos de link <a href="EnlaceQueApunta"> y <a name="AnclaQueApunta">
href="EnlaceQueApunta" Sirve para que el link apunte a otra pagina web o archivo html.
name="AnclaQueApunta" Sirve para que el link apunte a una seccion concreta dentro de la misma pagina web o archivo html.
Nota, para usar una etiqueta que apunta a un ancla, en la ruta hay que incluir el simbolo # antes del nombre del ancla.
Ademas, la etiqueta a tiene otro atributo que se conoce como target que sirve para elegir donde abrir el link, si en la misma pestaña o en una nueva pestaña.
<a href="EnlaceQueApunta"> por defecto abre el link en la misma pestaña.
<a href="EnlaceQueApunta" target="_blank"> abre el link en una nueva pestaña.

Ir al final de una web muy larga

Por ejemplo, si tenemos una pagina web con mucho texto y queremos llegar al final de la pagina sin tener que hacer scroll,
podemos crear un ancla al final de la pagina y un link al principio de la pagina que apunte a ese ancla.
