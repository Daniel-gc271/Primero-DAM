Emmets
Caracteres de control

Los emmets son palabras reservadas del ide que permiten diseñar o programar de manera mucho mas rapida.

Existen infinidad de ellos ">" "+" "*" "^" "$" "lorem"

Tambien existen caracteres de control "{}" "[]" "()" "@" "-" "#"

Índice
Emmets
Emmet#01 ">"
Sirve para crear una etiqueta hija (un nivel inferior)
Emmet#02 "+"
Sirve para crar una etiqueta hermana (mismo nivel)
Emmet#03 "*"
Sirve para realizar un emmet varias veces
Emmet#04 "^"
Sirve para subir un nivel
Emmet#05 "$"
Sirve para enumerar automáticamente
Emmet#06 "lorem"
Sirve para generar texto aleatorio de relleno
Emmet#07 "picsum"
https://picsum.photos sirve para obtener fotos placehorlder del tamaño que se quiera.
para escoger un tamaño basta con agregar/L/H/

Caracteres de control
Caracter de control#01 "{}"
Sirve para incluir texto en el interior de etiquetas
Caracter de control#02 "[]"
Sirve para incluir atributos en el interior de la etiqueta
Caracter de control#03 "()"
Sirve para agrupar emmets por ejemplo para usar el emmet "*"
Caracter de control#04 "@"
Sirve para indicar al emmet "$" desde que numero comenzar
Caracter de control#05 "-"
Sirve para indicar al emmet "$" que la cuenta es decreciente
Caracter de control#06 "#"
Sirve para agregar directamente un id a la etiqueta
