Explicacion Etiquetas de Formato y Estilo
Etiqueta#001 <hr>
Sirve para poner saltos de linea con una barra divisoria en horizontal

Etiqueta#002 <br>
Sirve para poner saltos de linea comunes y corrientes

Etiqueta#003 <h$>
Sirve para poner titulos, cuanto mayor sea el numero que la acompaña menor sera el tamaño del texto

Etiqueta#004 <p>
Sirve para delimitar parrafos

Etiqueta#005 <b>
Sirve para poner texto en negrita

Etiqueta#006 <i>
Sirve para poner texto en cursiva

Etiqueta#007 <u>
Sirve para subrayar texto

Etiqueta#008 <mark>
Sirve para destacar texto con un fluorescente

Etiqueta#009 <s>
Sirve para tachar texto

Etiqueta#010 <sup>
Sirve para poner superindices

Etiqueta#011 <sub>
Sirve para poner subindices

Etiqueta#012 <small>
Sirve para poner texto pequeño

Etiqueta#013 <code>
Sirve para poner texto en formato de codigo

Etiqueta#014 <fieldset>
Sirve para englobar partes de la pagina web en un recuadro;
Va acompañada de la etiqueta legend para poner un titulo al marco
